## Instructions for local use

### Installation

1. -- Clone server

```sh
git clone https://github.com/LukeAnton/UI-Challenge-Server.git
```

start server

```sh
   npm run json:server
```

2. -- Clone frontend

```sh
 git clone https://github.com/LukeAnton/UI-Challenge-Server.git
```

install packages npm install

```sh
  npm install or yarn
```

start server

```sh
   npm run start or yarn start
```
